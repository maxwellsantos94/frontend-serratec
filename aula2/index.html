Dia 3 — Introdução ao CSS

Seletores, propriedades de texto, cores, bordas e sombras.

As 3 formas de usar CSS

1. Externo (recomendado)

Arquivo .css separado vinculado no <head>.

<link rel="stylesheet" href="estilos.css">

2. Interno

Estilos dentro da tag <style> no <head>.

<style> body { background-color: #f0f0f0; } </style>

3. Inline

Estilo aplicado diretamente no atributo da tag HTML.

Exemplo de estilo inline aplicado aqui.

Seletores: Tag, Classe e ID

O seletor determina quais elementos HTML serão afetados pelas regras declaradas.

/* Tag: afeta todos os <p> */ p { color: #222; } /* Classe: afeta múltiplos elementos */ .destaque { background-color: yellow; } /* ID: afeta um elemento único */ #titulo { font-size: 2rem; }

Este parágrafo utiliza o seletor p.destaque.
